# dart-lab
Projeto de Estudo da Linguagem Dart

[ Progama de Estudo ]R

Na fase inicial o meu objetivo é aprender os conceitos basicos da linguagem
como 
- Variaveis
- Funcóes
- Estruturas de Controle
- Estruturas de Repetição
- Arrays
- Orientação a Obejetos
- Manipulação de Strings

e para facilitar vou estudar na linha de comando para não ter que
me preucupar com configurações de Servidor Web, Interação com HTML e etc.

### Preparando ambiente para rodar em Linha de Comando WINDOWS #####

INSTALAÇÃO DO DART-SDK
choco install dart-sdk

COMPILANDO
dart2aot src/main.dart

RODANDO BINARIO 
dartaotruntime bin/main.dart.aot

## DIRETORIOS ### 
/src => Diretorio para Arquivos Fonte
/bin => Diretorio de Arquivos Compilados
